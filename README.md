# Calculadora de Bitola

## Descrição do Projeto

No contexto de instalações elétricas, é fundamental dimensionar corretamente a bitola dos fios para garantir segurança, eficiência e evitar sobrecargas.

Este projeto foi desenvolvido a partir de uma proposta do professor de instalações elétricas, com o objetivo de auxiliar no cálculo da bitola adequada dos condutores elétricos.

O aplicativo recebe como entrada:
- Corrente elétrica (em ampères)  
- Distância do circuito (em metros)  

E retorna:
- Bitola do fio (em mm²) para tensão de 110V  
- Bitola do fio (em mm²) para tensão de 220V  

---

## Tecnologias Utilizadas

- Flutter  
- Dart  
- Material Design  

---

## Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/calculadora-bitola.git
```

2. Acesse a pasta do projeto:
```bash
cd calculadora-bitola
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Execute o aplicativo:
```bash
flutter run
```

---

## Funcionamento do Aplicativo

1. Insira o valor da corrente elétrica (A)  
2. Informe a distância do circuito (m)  
3. O sistema calculará automaticamente:
   - Bitola recomendada para 110V  
   - Bitola recomendada para 220V
